## 技术要点

### 唱片设计
- CSS `conic-gradient` 模拟黑胶唱片纹理（墨色基调 `#1a1410`/`#2a2218`）
- 朱砂红圆心标签（`#C41E1E`），阴刻"乐"字
- 播放时 CSS `@keyframes spin` 旋转动画（`animation-play-state` 切换）
- 悬停放大效果

### 播放器交互
- 使用 HTML5 `<audio>` 元素，零额外依赖
- 点击唱片展开/收起播放列表面板（`<transition>` 动画）
- 面板标题"山海乐坊"，列表显示序号+歌名+艺人
- 当前播放歌曲高亮 + ♪ 脉冲动画
- 自动切歌：`onended` → `next()` 循环播放
- 音频加载失败静默处理

### 面板样式
- 宣纸底色 `rgba(245, 240, 232, 0.96)` + 墨色文字
- 朱砂红主色调按钮（播放键）
- 圆角边框 + 阴影，古风克制
